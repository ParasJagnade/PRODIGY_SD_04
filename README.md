# Sudoku Solver - Java

This project is Task-04 for Prodigy Infotech Internship.

## Description
A Java program that solves a Sudoku puzzle using the Backtracking algorithm.

## Features
- Uses recursion and backtracking
- Ensures Sudoku rules are followed
- Displays solved grid

## How to Run
1. Compile:
   javac SudokuSolver.java
2. Run:
   java SudokuSolver

## Technology
- Java
